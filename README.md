# AI Code Debugger App

AI Code Debugger App is an AI-powered debugging assistant built with Streamlit, Gemini API, and Groq API.  
Users can upload screenshots of programming errors or code snippets, and the application analyzes them to provide either debugging hints or complete solutions.

Because apparently developers now debug code by taking screenshots of errors instead of reading stack traces. Evolution is fascinating.

---

# Features

- Upload up to 2 code/error screenshots
- AI-based code analysis using Gemini
- Two response modes:
  - Hints Mode
  - Full Solution Mode
- Follow-up AI chat assistant using Groq
- Maintains chat history
- Clean Streamlit UI
- Multi-image support for better debugging context

# How It Works

## Step 1: Upload Images

Users upload screenshots containing:
- Error messages
- Code snippets
- Console outputs

Maximum 2 images are allowed.

---

## Step 2: Select Response Type

User can choose:

### Hints
- Step-by-step debugging guidance
- No direct code solution

### Solution
- Complete fixed code
- Explanation of the issue

---

## Step 3: Gemini Analysis

The uploaded images are sent to Gemini AI for:
- Error detection
- Code analysis
- Bug explanation
- Solution generation

---

## Step 4: Follow-up Chat

After generating the response, users can ask additional questions using Groq AI.

Example:
- "Why this error happened?"
- "Can you optimize this code?"
- "Explain this line"

Humanity finally invented recursive suffering:
AI explains code generated by another AI.
